# ScriptSchedule

Transform your handwritten to-do lists into intelligent digital schedules with AI-powered OCR and smart scheduling.

## Setup & Installation

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd goodnotes-hackathon
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open in browser**
   - The app will automatically open at `http://localhost:3000`
   - If it doesn't open automatically, navigate to `http://localhost:3000` in your browser

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App (not recommended)

## Quick Start

1. **Upload Demo**: Click "Try Demo" to see the app in action with sample data
2. **Upload Image**: Drag and drop or select a handwritten to-do list image
3. **Review Tasks**: Edit extracted tasks as needed
4. **Schedule**: Add tasks to your calendar with smart scheduling
5. **Manage Events**: Click calendar events to view/edit details

## Features

- 📝 **AI-Powered OCR**: Extract tasks from handwritten notes
- 📅 **Smart Scheduling**: Intelligent calendar integration
- 🤖 **AI Agent**: Automated conflict resolution and optimization
- 📚 **Course Suggestions**: Learning recommendations for skill development
- 📱 **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

- **Frontend**: React 18, Tailwind CSS
- **Icons**: Lucide React
- **Date Handling**: date-fns
- **Styling**: Custom GoodNotes-inspired design system
- **Demo Data**: Comprehensive mock services (no backend required)

## Demo Mode

The app runs entirely with mock data for demonstration purposes. All features are fully functional without requiring any backend services or API integrations.
